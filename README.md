# Recriando o Wikipedia com Layout Moderno

Projeto desenvolvido como desafio do módulo 3 da trilha de HTML da DIO.

## Objetivo

Criar uma página inspirada na Wikipedia, utilizando HTML semântico, estrutura moderna e recursos básicos de acessibilidade.

## Tema escolhido

O tema escolhido para a página foi:

**Java - Linguagem de Programação**

A escolha foi feita por ser um assunto relacionado à área de Análise e Desenvolvimento de Sistemas e ao estudo de programação.

## Recursos utilizados

O projeto utiliza:

- HTML5
- CSS3
- Estrutura semântica
- Layout moderno
- Menu lateral
- Navegação por âncoras
- Tabela
- Imagem com texto alternativo
- Citações
- Links externos
- Responsividade básica
- Recursos de acessibilidade

## Tags semânticas utilizadas

- header
- nav
- main
- article
- section
- aside
- footer
- figure
- figcaption
- blockquote
- table
- caption
- thead
- tbody

## Recursos de acessibilidade

Foram utilizados alguns recursos para melhorar a acessibilidade da página:

- Atributo `alt` em imagem;
- `aria-label` em menus e áreas complementares;
- Hierarquia correta de títulos;
- Uso de `caption` em tabela;
- Uso de `scope` em cabeçalhos de tabela;
- Links com texto descritivo;
- Estrutura semântica clara.

## Como visualizar

Abra o arquivo `index.html` no navegador.
